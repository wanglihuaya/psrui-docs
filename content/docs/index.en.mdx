Welcome to the PSRUI documentation site.

This site combines four kinds of material in one place:

- practical instructions for opening archives, previewing charts, processing data, and exporting results in PSRUI
- compact explanations of pulsar-processing concepts such as DM, dedispersion, folding, TOA extraction, RFI, and calibration
- a mirrored `Pulsar Physics` section that follows the same chapter structure as the Chinese `脉冲星物理学` pages
- upstream toolchain notes for PSRCHIVE, `paz`, `pam`, `pat`, `pac`, `tempo2`, and `dspsr`

## Suggested reading order

1. Start with [Getting Started](./getting-started) to choose a runtime and complete a first archive workflow.
2. Continue with [PSRUI Guide](./psrui-guide) for session previews, the Processing Inspector, batch recipes, and current feature boundaries.
3. Read [Pulsar Basics](./pulsar-basics) when you want more context behind the terminology.
4. Continue with [Pulsar Physics](./pulsar-physics) if you want the English section that mirrors the Chinese `脉冲星物理学` directory.
5. Continue with [Handbook of Pulsar Astronomy](./handbook-pulsar-astronomy) only if you want a separate handbook-style English reading path from a different source.
6. Use [Toolchain Reference](./toolchain) when you need to connect the GUI workflow to upstream CLI tools.

## How bilingual content works here

- Chinese is the default route at `/docs/...`
- English mirrors live at `/en/docs/...`
- The language icon at the bottom-left of the sidebar switches to the matching page in the other locale whenever possible
