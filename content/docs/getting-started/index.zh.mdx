## 这一节解决什么问题

- 这个应用适合谁，不适合谁
- 本机 `local` runtime 和 Docker runtime 该怎么选
- 从“打开 archive”到“导出处理结果”的最短路径是什么

> **PSRUI 当前支持**：archive 预览、channel zapping、`pam` 参数预览、`pat` TOA 提取、`pac` calibration preview、batch recipe。

> **常见误解**：PSRUI 当前还不是完整的 `tempo2` timing residual GUI，也不是 raw baseband 到 folded archive 的 `dspsr` 前端。
