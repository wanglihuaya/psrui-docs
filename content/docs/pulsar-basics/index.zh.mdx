这部分不是完整天体物理教材，而是给 PSRUI 用户准备的一组“操作前先懂一点”的概念页。

它的目标不是替代教材，而是先回答三个更现实的问题：

- 你在 GUI 里看到的术语，分别落在处理链的哪一步
- 为什么某些按钮会改变 profile、waterfall 或偏振图的样子
- 什么时候该继续留在 PSRUI，什么时候应该切回更完整的命令行时序分析流程

建议先读：

- [Archive、DM 与 TOA](./archive-dm-toa)
- [RFI 与 calibration](./rfi-calibration)
- [脉冲星物理学](../pulsar-physics)
- [Handbook of Pulsar Astronomy](../handbook-pulsar-astronomy)

## 这一节怎么用

如果你已经开始上手 PSRUI，推荐这样读：

1. 先用 [Archive、DM 与 TOA](./archive-dm-toa) 把“数据是什么、为什么要对齐、TOA 从哪里来”串起来。
2. 再用 [RFI 与 calibration](./rfi-calibration) 把“数据污染”和“仪器响应修正”区分开。
3. 如果你还想沿着教材章节往后读，再转到 [脉冲星物理学](../pulsar-physics)。
4. 如果你想沿着“观测设备 -> 搜寻 -> 已知源观测 -> 计时”的 handbook 逻辑继续读，再转到 [Handbook of Pulsar Astronomy](../handbook-pulsar-astronomy)。

## 为什么这里仍然需要基础概念页

图形界面会让操作更直观，但不会自动消除术语之间的依赖关系。

如果你不知道：

- DM 为什么影响频率对齐
- folding 为什么能把周期信号累积出来
- TOA 为什么依赖 template
- calibration 为什么会改变 Stokes 相关表现

那么你即使点得动 GUI，也不一定能解释结果。

## 和“脉冲星物理学”分区的关系

- 这里强调“处理动作和结果怎么理解”，优先服务 PSRUI 的使用场景。
- [脉冲星物理学](../pulsar-physics) 则按教材目录展开，适合系统读发现、观测、时间特性、辐射与演化。
- [Handbook of Pulsar Astronomy](../handbook-pulsar-astronomy) 则补上搜寻、观测、计时和仪器链路的 handbook 视角。
- 三者分别对应“快速上手”“中文章节导读”“观测方法 handbook”，不是重复内容。
