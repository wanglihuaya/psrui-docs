欢迎来到 PSRUI 文档站。

这里把五类信息放到同一个入口里：

- 如何使用 PSRUI 打开 archive、预览图表、做非破坏处理与导出
- 脉冲星数据处理中最常见的核心概念，例如 DM、dedispersion、folding、TOA、RFI 与 calibration
- 按《脉冲星物理》章节整理的系统背景阅读，帮助你把 GUI 操作放回更完整的物理图景里
- 按 *Handbook of Pulsar Astronomy* 重写的中英文 handbook 导读，补上观测、计时、搜寻与仪器链路
- 与 PSRUI 最相关的上游工具链，包括 PSRCHIVE、`paz`、`pam`、`pat`、`pac`、`tempo2`、`dspsr`

## 建议阅读路径

1. 从 [入门](./getting-started) 开始，先把安装方式、runtime 选择和第一条处理路径走通。
2. 再读 [PSRUI 使用](./psrui-guide)，理解 session preview、Processing Inspector、batch recipe 和当前能力边界。
3. 如果你需要先补术语和处理链路，再读 [脉冲星基础](./pulsar-basics)。
4. 如果你想按教材脉络系统梳理，再读 [脉冲星物理学](./pulsar-physics)。
5. 如果你想要一本更偏观测方法论、同时兼顾中英文切换的 handbook 式路径，再读 [Handbook of Pulsar Astronomy](./handbook-pulsar-astronomy)。
6. 当你要把图形界面和上游命令行工作流对应起来时，再读 [工具链参考](./toolchain)。

## 长篇阅读分区怎么选

- [脉冲星基础](./pulsar-basics) 先解释 archive、DM、TOA、RFI 与 calibration 这些 GUI 最常见术语。
- [脉冲星物理学](./pulsar-physics) 更像中文教材章节导读，按发现、观测、时间特性、演化与星际介质展开。
- [Handbook of Pulsar Astronomy](./handbook-pulsar-astronomy) 更偏 handbook 风格，强调观测流程、搜寻、计时、仪器与跨波段视角。

## 本站如何组织双语内容

- 中文是默认入口，路径保留在 `/docs/...`
- 英文镜像位于 `/en/docs/...`
- 左侧底部的语言 icon 会优先切换到当前页面的另一语言版本
